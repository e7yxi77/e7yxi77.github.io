DS-Server rulesㅣ서버 규칙

  • This server is a Survival Server. There are no commands allowed during the playtime. (However, administrators are allowed to use commands during server maintenance and bug fixes.)
  • Everyone should be able to enjoy the server. At the first day of the server opening, there is no PVP and stealing allowed
  • No hacks allowed. → Including auto-build AI, X-ray.

  • Boss monsters should be defeated with all players on the server. → This will be recorded in the server log.

  • If you don't follow the rules: 2 warnings → Permanent ban from the server.

  • 이 서버는 치트가 없는 서버입니다. 어떤 상황이든 모든 치트는 금지됩니다. (단, 관리자가 서버 점검 및 버그 수정시 커맨드 사용 가능)
  • 모든 사람이 서버를 즐길 수 있어야 합니다. 첫날에는 게임을 즐기기 위해 PVP 및 약탈은 금지됩니다
  • 해킹은 금지됩니다. → 자동 건물 AI, X-레이를 포함합니다.

  • 보스 몬스터는 서버에 있는 모든 플레이어와 함께 처치되어야 합니다. → 이는 서버 로그에 기록됩니다.

  • 위 규칙을 위반시 2번의 경고 후 → 영구적으로 서버에서 차단될 수 있습니다.
